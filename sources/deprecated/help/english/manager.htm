Cracklock Manager

Cracklock Manager is an utility which lets you add and configure Cracklock controlled programs. All programs that Cracklock supervises are placed in a list helded by the manager.


The manager

The manager interface is very easy to use, as it looks like Windows Explorer. This tools assists you in the 3 main functions :

  • adding a new program,
  • setting date and time parameters for this program,
  • executing the program (though you are not obliged to do so from the manager, as the program may be straight executed).
