Advice for users of previous versions of Cracklock

  • If a previous version of Cracklock is installed on your computer, it is better to suppress it before installing the new version.
  • If you have a version before the version 2.3 : before suppressing it , you must first disable Cracklock for all the programs for which it was enabled. Then, you can uninstall Cracklock from the configuration panel of Windows.
