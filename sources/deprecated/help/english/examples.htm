Examples included with the distribution files of Cracklock

Cracklock is distributed together with two examples showing date and time. To learn using Cracklock or to check its functions, you can try to parameter these examples by using the manager.

These two examples are located in the folder "Examples" in the directory where you installed Cracklock.

File Description Programming language
vcdate.exe DigClock program from the book Programming Windows 95 by Charles Petzold (Microsoft Press) Visual C++
vbdate.exe A little program of my own. Visual Basic 5.0
