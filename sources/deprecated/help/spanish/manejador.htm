Manejador de Cracklock

El Manejador de Cracklock es una utilería que le permite agregar y configurar los programas controlados por Cracklock. Todos los programas que Cracklock supervisa son puestos en una lista mantenida por el manejador.


El manejador

La interface del manejador es muy fácil de usar pues tiene un parecido con el Explorador de Windows.

esta herramienta le facilita las 3 funciones fundamentales :

• Agregar un nuevo programa

• Configurar los parámetros de hora/fecha para este programa

• Ejecutar el programa (si bien usted no está obligado a hacerlo desde el manejador, pues el programa se puede ejecutar directamente ).
