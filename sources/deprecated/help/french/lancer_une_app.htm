Lancer l'application

Pour lancer un application sous le contrôle de Cracklock, vous pouvez soit la lancer normalement en passant par son raccourci dans le menu "Démarrer" soit en sélectionnant l'application dans le gestionnaire et en cliquant sur le menu Exécuter.

 

Cas particulier :

Si vous avez choisi dans l'onglet de dépendance de ne pas modifier les fichiers de votre application, vous devez alors la lancer différemment. C'est à dire soit :

" En utilisant le gestionnaire, de la même manière que d'habitude

" En modifiant le raccourci de votre application : faites pointer le raccourci de l'application (situé dans le menu Démarrer de Windows 95) sur la commande suivante :

C:\PROGRA~1\Cracklock\MCL.EXE C:\APP\APP.EXE [vos paramètres]

où

- C:\PROGRA~1\Cracklock\MCL.EXE est le chemin d'accès au programme MCL.EXE installé par Cracklock

- C:\APP\APP.EXE est le chemin d''accès au fichier EXE de votre applicationexecutable file)

" En utilisant le menu contextuelle de l'explorer de Windows

Dans l'explorer de Windows, aller dans le dossier où se trouvent les fichiers de votre application


Explorer de Windows

 

Dans cette fenêtre, cliquez avec le bouton droit sur le ficher exécutable de votre application (ici VBDate.exe) . Un menu contextuelle apparait :


Menu contextuel

 

Enfin sélectionnez le menu "Démarrer avec Cracklock".
