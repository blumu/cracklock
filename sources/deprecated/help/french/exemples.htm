Les exemples fournis aves Cracklock

Cracklock est fournis avec deux exemples de logiciels affichant la date et l'heure. Pour apprendre à utiliser Cracklock ou pour vérifier qu'il fonctionne bien, vous pouvez essayer de configurer ces examples en utilisant le gestionnaire.

Ces deux examples se situent dans le sous-dossier "Examples" du dossier où vous avez installez Cracklock.
