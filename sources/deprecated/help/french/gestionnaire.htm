Le Gestionnaire de Cracklock

Le gestionnaire de Cracklock est un utilitaire qui vous permet d'ajouter et de configurer les programmes contrôlés par Cracklock. Tous les programmes mis sous la surveillance de Cracklock se trouvent dans une liste maintenue par le gestionnaire.


Le gestionnaire

L'interface du gestionnaire est très simple du fait de sa similarité avec l'explorer de Windows.

Cet outils vous assiste dans les 3 opérations principales :

" ajouter une nouvelle application

" configurer la date et l'heure de cette application

" lancer l'application (bien qu'il ne soit pas obligatoire de le faire à partir du gestionnaire.)
