Disfonctionnements

" Sous Windows NT, Cracklock n'est pas capable de modifier des fichiers systèmes telle que MSVCRT.DLL.

" La version courrante de Cracklock ne reconnait pas les programmes configurés par les versions 1.x et 2.x de Cracklock.

" Cracklock ne peut pas fonctionner sur des applications 16 bits (pour Windows 3.1)
