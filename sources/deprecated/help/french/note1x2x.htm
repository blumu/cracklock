Note aux utilisateurs de version précédentes de Cracklock :

" Si votre ordinateur contient une version antérieure de Cracklock, il est conseiller de la supprimer puis d'installer la nouvelle version.

" Si vous possédez une version antérieure à la version 2.3 : avant de supprimer cette version, vous devez d'abord désactiver Cracklock pour toutes les applications sur lesquels il est actif. Ensuite, vous pouvez désinstaller Cracklock à partir du panneaux de configuration de Windows.
